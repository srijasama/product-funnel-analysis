# Customer Funnel Analysis for E-commerce Behavior

## Project Overview
This project analyzes customer behavior in an e-commerce dataset using Python and Pandas to understand how customers progress through different engagement stages.

## Funnel Stages
- Total Customers
- Repeat Buyers (10+ items purchased)
- High-Value Customers ($1000+ spend)
- Discount Users

## Tools Used
- Python
- Pandas
- Matplotlib
- Google Colab

## Key Insights
- Only a portion of customers become repeat buyers.
- High-value customers represent a smaller segment but contribute significantly to revenue.
- Discounts are used by only part of the customer base.

## Business Recommendations
- Target medium-engagement customers with loyalty incentives.
- Encourage repeat purchases through personalized marketing.
- Use discount campaigns strategically to increase engagement.
